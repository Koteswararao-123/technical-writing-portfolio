# How to Install Python on Windows


## Introduction
Python is a programming language that is easy to learn. This guide shows how to install Python on a Windows system step by step.


## Prerequisites
- Windows system
- Internet connection
- Web browser (Chrome, Edge, etc.)


## Steps to Install Python
1. Open a web browser.
2. Go to https://www.python.org
3. Click on **Download Python**.
4. Open the downloaded installer file.
5. Select the checkbox **Add Python to PATH**.
6. Click **Install Now**.
7. Wait for installation to complete.
8. Click **Close**.


## Verify Installation
1. Open Command Prompt.
2. Type: python --version
3. Press Enter.
4. If the Python version is displayed, installation is successful.


## Common Issues
**Python not recognized**
- This happens if PATH is not selected.
- Reinstall Python and select **Add Python to PATH**.

**Installation failed**
- Run the installer again as administrator.


## Conclusion
Python installation is simple and takes only a few minutes. You can now start using Python.
