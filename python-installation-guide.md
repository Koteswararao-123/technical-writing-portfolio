# How to Install Python on Windows


## Introduction
Python is a programming language that is easy to learn. This guide shows how to install Python on a Windows system step by step.


## Prerequisites
- Windows system
- Internet connection
- Web browser (Chrome, Edge, etc.)


## Steps to Install Python
1. Open a web browser.
2. Go to https://www.python.org
3. Click on **Download Python**.
4. Open the downloaded installer file.
5. Select the checkbox **Add Python to PATH**.
6. Click **Install Now**.
7. Wait for installation to complete.
8. Click **Close**.


## Verify Installation
1. Open Command Prompt.
2. Type: python --version
3. Press Enter.
4. If the Python version is displayed, installation is successful.


## Troubleshooting
If you face issues during installation, try the following:
- Restart your system and run the installer again  
- Make sure you have a stable internet connection  
- Disable antivirus temporarily if it blocks installation  
- Run the installer as Administrator  
If the problem still continues, try downloading the installer again from the official website.


## Common Errors
**Python is not recognized as an internal or external command**
- This error occurs when Python is not added to PATH  
- Reinstall Python and select “Add Python to PATH”  

**Installation failed**
- Close all programs and try installing again  
- Run the installer as Administrator  

**Permission denied error**
- You may not have sufficient permissions  
- Try running the installer with admin rights

  
## Common Issues
**Python not recognized**
- This happens if PATH is not selected.
- Reinstall Python and select **Add Python to PATH**.


## Notes
- Always download Python from the official website  
- Make sure to select “Add Python to PATH” during installation  
- Use the latest version of Python for better features and support

    
**Installation failed**
- Run the installer again as administrator.


## Conclusion
Python installation is simple and takes only a few minutes. You can now start using Python.
